# Poke Robot

> 定义一款可以走进千家万户、真正能干活的 Home Robot。

## 公司三条主线

| 主线 | 说明 |
|------|------|
| **AI** | 核心智能能力 |
| **产品** | 产品定义 & 工业设计 |
| **生态 / PR** | 品牌建设、内容传播、创始人IP |

## 项目结构

```
Poke Robot/
├── website/         # 官方网站（Responsive）
│   ├── assets/      # 网站专用素材
│   ├── src/         # 源代码
│   └── public/      # 静态资源
├── docs/            # 文档中心
│   ├── brand/       # 品牌（Logo、VI、调性）
│   ├── product/     # 产品定义 & 工业设计
│   ├── pr/          # PR策略、个人IP运营
│   ├── workspace/   # 办公空间 & 软装
│   └── planning/    # 工作规划 & 里程碑
└── assets/          # 公司级公共素材库
    ├── logo/
    ├── video/
    ├── photo/
    ├── design/
    └── icon/
```

## 近期关键节点

- **3月底~4月初**：PR 短片发布（创始人个人账号）
- **同期完成**：公司 Logo、官网、公众号等对外内容上线
- **持续进行**：办公空间居家风格软装、工业设计师招聘
