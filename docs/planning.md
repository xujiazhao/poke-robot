# Poke Robot — 品牌与产品策略思考

> 更新：2026-03-14 | 作者：设计顾问

这份文档是我对 Poke Robot 品牌、产品和对外表达的系统性思考。目的是把想法理清楚，和创始人对齐，然后再拆解成可执行的任务。

---

## 一、我怎么理解这家公司

Poke Robot 现在的状态是：一群工程师想做一件改变世界的事，但还没有找到那个"一句话让所有人明白你在做什么"的表达。

三条主线（AI、产品、生态/PR）是内部组织逻辑，但对外面的人——用户、媒体、投资人、未来的候选人——他们只想知道一个问题：**你们做的东西，跟我有什么关系？**

我的判断是：这家公司的核心故事不是"我们技术多强"，而是 **"我们要让机器人真正走进你家"**。这个故事的力量在于它非常直觉、非常情感化，而且市面上没有人真的做到了。

---

## 二、产品的本质——为什么必须像人

创始人说了一句很精准的话：**如果一个机器人需要长得像生物，它一定有特定的情绪意义，不然完全可以是机器的外形。**

这句话其实定义了产品哲学的根基。我展开来说：

### 扫地机器人解决的是问题，Poke 解决的是关系

扫地机器人是工具。它的价值公式很简单：省时间 = 有用。所以它长得像个冰球没关系，你不需要跟它有情感连接。

但 Poke 要做的事情不一样。一个在家里走来走去、帮你干活的类人型机器人，用户天然会用"相处"的心态去面对它——它不是一个放在角落充电的设备，它是一个会出现在你视野里的**存在**。

这就意味着：
- **外观**不是工业设计问题，是**心理预期管理**。类人外形给用户的隐性承诺是"我可以像对人一样对你"。
- **人格模拟**不是花哨功能，是**使用体验的底层架构**。如果它长得像人但行为像机器，反而会让人不舒服（恐怖谷效应的另一面）。
- **能干活**是最基本的尊重。如果它长得像人却什么都不会做，那就是一个昂贵的表演道具。

### iPhone 时刻的真正含义

创始人说要做**机器人行业的 iPhone 时刻**。iPhone 成功不是因为它是最好的手机，而是因为它第一次让普通人觉得"这个东西我必须拥有"。

对 Poke 来说，iPhone 时刻 = **从"哇，好酷"到"我家需要一个"的跨越**。

这个跨越要靠三件事：
1. **有用**（不是表演，是真的能完成具体任务）
2. **不吓人**（亲切、温暖、可预期）
3. **买得起**（千家万户 = 大众价格）

## 三、品牌表达——对外怎么说

### 核心 Tagline（候选）

| 方案 | 调性 | 适合场景 |
|------|------|----------|
| Not a machine. A companion that works. | 克制、有力 | 官网 Hero、品牌片 |
| The robot your home has been waiting for. | 温暖、期待感 | PR 传播、社媒 |
| Built to belong. | 极简、记忆点强 | Logo 旁边、slogan |

建议 **"Built to belong."** 作为最精炼的品牌 slogan，其他两句在不同场景使用。

### 三条产品哲学

我把产品的核心belief提炼成三句话，每句话对应一个设计决策的"为什么"：

**01 — 形态即态度（Form is intention）**
> 选择类人外形不是为了炫技，而是因为家需要一个"人"——不是一台机器。如果它长得像生物，那它就承载了情绪意义。

**02 — 能力即尊重（Utility is respect）**
> 能干活，是对用户时间的尊重。不表演，是对用户智商的尊重。

**03 — 人格即连接（Personality is connection）**
> 模拟人格不是欺骗，是让人与机器人之间的相处自然发生。温暖不是一个功能，是一个设计原则。

### Brand Manifesto

> "We don't build robots that perform. We build robots that *belong*."

这句话的力量在于 perform vs belong 的对比——perform 暗示那些翻跟头、跳舞的表演机器人；belong 暗示"家庭成员"。一句话就把 Poke 和所有竞品拉开了距离。

## 四、创始人 IP 的思路

创始人的个人 IP 要和公司深度绑定，但视觉上要更亲切、更有朋友感。

我的思路是：**公司品牌是"理想"，创始人 IP 是"过程"。**

- 公司品牌讲的是"我们要做什么" — 宏大、有信念感、质感偏高
- 创始人 IP 讲的是"我们怎么做的" — 真实、有烟火气、Build in Public

两者不矛盾，反而互补：看了公司品牌觉得"这个目标好大"，看了创始人日常觉得"这些人真的在一步步做"。

内容形式建议：
- **播客**：技术思考 + 创业故事，偏深度
- **短视频/vlog**：办公室日常、机器人调试过程偏轻松
- **年度总结**：Build in Public 的标志性内容格式
- **视觉风格**：和公司主品牌共用色系，但更手绘/更有温度，比如手写字体、随手拍质感

## 五、对工业设计的初步想法

还没招到工业设计师，但有几个方向我觉得值得提前对齐：

1. **不要"科幻感"**：走进千家万户的东西不能让人觉得"这是个实验品"。参考路径：Dyson 不是科幻感，是"高级家电感"。
2. **材质要有温度**：金属 + 硬塑料 = 冷冰冰。可以考虑织物包裹、圆润曲线、哑光质感。
3. **尺度感很重要**：太大会有压迫感，太小会觉得像玩具。需要找到"家庭成员"的合适体量。
4. **面部/头部是关键**：这是用户建立情感连接的第一触点。不需要高仿真人脸，但需要一个可以让人"读出表情"的界面。

---

## 六、办公空间的想法

居家风格的办公室，同时兼顾机器人调试场景——这其实本身就是一个很好的品牌叙事素材。

建议：
- 软装调性和品牌视觉保持一致（暖色调、自然材质、有生活感）
- 留出"可拍摄区域"：办公室本身就是内容场景（Build in Public 的背景、PR短片取景）
- 机器人调试区自然融入生活场景，而不是单独一个"实验室" — 这本身就在说"我们的机器人属于家"
